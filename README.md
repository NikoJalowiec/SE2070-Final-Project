## Final Project
A very simple weather app that provides current weather data and a 7 day forecast.

Features:

    Choice between Imperial and Metric measurments

    Current weather report containing temperature, feels like temperature, windspeed, and a general description.

    7 day forecast containing temperature highs and lows, general descriptions, wind speed, and chance of rain.

Instructions:
    Download all the files into a folder, using a compiler of your choice compile the java files and then run. You will be prompted to enter a zipcode and can choose which measurment system you'd like to use. Once you hit enter the app will retrieve the current weather for that zipcode. From there you can refresh the page to get updated information or click the Forecast button to get a 7 day forecast for the area.  You can also return to the current weather.
